Project Eleven

Open-Source Quadrupedal Robot

Project Eleven is an open-source quadrupedal robot designed to explore dynamic motion, real-time sensor feedback, and control optimization in both simulated and physical domains.

The project currently focuses on building a robust simulation in MuJoCo, where we test control strategies and analyze joint, IMU, and force data. We're working to stabilize the robot’s gait and understand its physical interaction models.

From August 2025, we begin constructing the actual robotic hardware — translating simulation insights into a real-world platform.
